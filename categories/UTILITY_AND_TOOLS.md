# Utility & Tools

Build practical applications and developer tools that solve real-world problems.

## 1. CLI Password Manager
Build a secure command-line tool to generate, store, and retrieve passwords with encryption and clipboard integration.

- **Difficulty:** Beginner to Intermediate
- **Tech Stack:** Python or Rust, cryptography library, SQLite, Click or Clap
- **Learning Outcomes:**
  - Encryption and hashing basics
  - CLI argument parsing
  - Secure credential storage
  - Operating system integration
- **Time Estimate:** 2-3 weeks

## 2. Static Site Generator
Create a tool that converts markdown files into a static HTML website with templating, asset optimization, and hot reload during development.

- **Difficulty:** Intermediate
- **Tech Stack:** Python or Node.js, Markdown parser, Jinja2/Handlebars, File watcher
- **Learning Outcomes:**
  - File system operations
  - Template engines
  - Build pipelines
  - Asset optimization
- **Time Estimate:** 3-4 weeks

## 3. API Rate Limiter Service
Build a distributed rate limiting service that throttles API requests based on user, IP, or endpoint with Redis backend.

- **Difficulty:** Intermediate to Advanced
- **Tech Stack:** Node.js or Python, Redis, Express or FastAPI, Docker
- **Learning Outcomes:**
  - Distributed systems concepts
  - Redis data structures
  - Rate limiting algorithms
  - Microservices architecture
- **Time Estimate:** 3-4 weeks

## 4. Automated Screenshot & Testing Tool
Build a tool that automatically takes screenshots of web pages, compares them visually to detect regressions, and alerts on changes.

- **Difficulty:** Intermediate
- **Tech Stack:** Puppeteer or Playwright, Node.js, Image processing library, CI/CD integration
- **Learning Outcomes:**
  - Headless browser automation
  - Image comparison algorithms
  - Testing pipeline integration
  - Screenshot diffing
- **Time Estimate:** 3-4 weeks

## 5. Local DNS Server with Ad Blocking
Create a lightweight DNS server that blocks ad domains locally and provides custom domain routing for development.

- **Difficulty:** Advanced
- **Tech Stack:** Go or Rust, DNS protocol knowledge, Socket programming
- **Learning Outcomes:**
  - Network protocols (DNS)
  - Low-level socket programming
  - Performance optimization
  - System administration concepts
- **Time Estimate:** 4-5 weeks

## 6. Backup & File Sync Tool
Build a tool to automatically backup files to cloud storage (S3, Google Drive) with versioning, compression, and scheduling.

- **Difficulty:** Intermediate
- **Tech Stack:** Python or Go, Cloud SDK (AWS/GCP), Cron jobs, Compression library
- **Learning Outcomes:**
  - Cloud API integration
  - File compression and deduplication
  - Scheduling and background jobs
  - Error handling and retry logic
- **Time Estimate:** 3-4 weeks

## 7. Code Snippet Manager with Search
Build a CLI tool to organize, tag, and search code snippets locally with syntax highlighting and export capabilities.

- **Difficulty:** Beginner to Intermediate
- **Tech Stack:** Python or Node.js, SQLite, Fzf or similar for search, Syntax highlighter
- **Learning Outcomes:**
  - Database design and queries
  - Full-text search implementation
  - File I/O operations
  - Interactive CLI design
- **Time Estimate:** 2-3 weeks

## 8. Log Analyzer & Alert System
Create a tool that parses application logs, detects patterns/anomalies, and sends alerts via email or Slack.

- **Difficulty:** Intermediate
- **Tech Stack:** Python or Node.js, Regex/pattern matching, Slack/Email API, Elasticsearch (optional)
- **Learning Outcomes:**
  - Log parsing and analysis
  - Pattern recognition
  - API integrations
  - Real-time alerting systems
- **Time Estimate:** 3-4 weeks
